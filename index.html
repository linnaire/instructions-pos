Creating Database

Open phpmyadmin

NOTE : In creating database, make sure you RUN your XAMPP or WAMPP.

In running XAMPP or WAMPP make sure that you have quit your SKYPE because it gets the PORT 80 used for APACHE.

For more details in running and configuring WAMPP click this link.

Step 1

Once you've configured your XAMPP or WAMPP, go to your browser and type or click this link localhost:80/phpmyadmin or localhost:8080/phpmyadmin

If the link doesn't work just reload it using the same URL.

Enter Credentials

Step 2

Once the phpmyadmin page is loaded, you will be redirected to their login page where you will input the credentials and click login.

NOTE : By Default the username for phpmyadmin is root and just leave the password text field empty.

Create Database

Step 3

Once you finished entering the credentials, you will be redirected to the dashboard page. And in there, you will find the New in a tree node. Click it to be redirected to Create Database page.

Now that you are in the database page, Type the database name blv_pos_db on the database text field and click Create.

CONGRATULATIONS! you have created the database for the project. Now you can proceed to the next step by clicking Cloning the project.

Cloning the Project

Running Command the Line

Step 1

In running the command prompt (cmd), Click the windows key and search for Command Prompt, then right click and Run as Administrator.

Switching Directories

Step 2

Now that you have run your Command Prompt as Administrator, We will now proceed to the next step, which is switching directories.

When we run the command prompt as administrator, our default directory is C:\Windows\system32, but we need our project to be in Desktop, so that it's accessible and easy to locate.

In order for us to switch the directory we need to type this command, cd C:/Users/lgalleno/Desktop.

The cd stands for "Change Directory".

The location of your desktop is in C:/Users/your_username/Desktop, In my case my user name is lgalleno but yours will be different. You can click this link for more information

Cloning the Project

VERY IMPORTANT NOTE : Make sure that you have installed your git.

In pasting LINKS on the Command prompt, Right-Click then Paste.

VERY IMPORTANT NOTE : Make sure that you have installed composer

Step 3

Now that you have successfully changed your directory into Desktop, you may now go to this link and COPY the url needed for cloning.

After copying the link, go to the Command Prompt again and type git clone url (Paste the link that you copied on bit bucket, Right click then "Paste" [example : git clone https://linnaire@bitbucket.org/vienzent/blv-pos.git]) then click "Enter"

After cloning, type this following commands.

  • cd blv-pos\
  • git config user.email "your_email" [example : git config user.email "vienzent0000@gmail.com"]
  • git config user.name "your_name" [example : git config user.name "tukmol"]
  • composer install
  • php -r "copy('.env.example', '.env');"
  • php artisan key:generate

CONGRATULATIONS! You have now finished cloning the project! You may now proceed to step 3, which is running the project!

Start Running the Project

VERY IMPORTANT NOTE : In running the project always make sure that your WAMPP is running and the wampp icon that is located in your task bar should be green.

NOTE : Make sure that the directory in your command prompt is in Desktop/blv-pos. If not, Re-run the Command Prompt as Administrator then change the directory to Desktop/blv-pos [ example : cd C:/Users/lgalleno/Desktop/blv-pos ]

STEP 1

After finishing the cloning, run this following commands :

  • php artisan migrate:refresh --seed
  • php artisan serve --port=8888

Once you have finished entering php artisan serve --port=8888 you will be redirected to this landing page where you will need to supply the credentials.
